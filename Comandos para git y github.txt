Git:
- git init (inicia un nuevo repositorio en la carpeta donde se abrio el git bash)
- git add -A (agrega todos los archivos al área de preparación)
- git status (opcional, sirve para ver los archivos en esa área)
- git commit -m "mensaje" (Hacer el commit de los cambios)
- git log (opcional, sirve para ver todos los commits)
Github:
- git push origin master (Subir los nuevos commits creados localmente hacia github)
- git fetch origin (Bajar los cambios de github hacia el directorio local)
- git merge origin/master (Fusionar la rama de los cambios bajados con la rama principal, es decir, aplicar los cambios bajados)
NOTA: Si hubo cambios en el repositorio local antes de bajarse los cambios desde github (con fetch origin) y si se desean descartar por los cambios nuevos de la página (github), aplicar el siguiente comando en vez del git merge: git reset --hard origin/master
